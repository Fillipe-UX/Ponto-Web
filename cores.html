Cor

A cor distingue nosso produto e nos ajuda a criar experiências consistentes em todos os produtos.

Cores saturadas

As cores saturadas infundem mais significado a experiência, destacam a UI, ajudam a criar associações com cores semelhantes.


Cores e suas funções

Os papéis das cores descrevem a intenção por trás da cor. Por exemplo, funções de cores são aplicadas a botões para diferenciar entre ações primárias, secundárias, de aviso ou perigosas.

Função e Cor Descrição
Neutros
Use para texto padrão e elementos secundários da interface do usuário, como botões secundários ou elementos de navegação.
Marca
Use para ações ou elementos principais que comunicam a marca do produto Ponto Web.
Descoberta
Use para interface do usuário que comunica algo novo, como integração ou informações sobre novos recursos.
Sucesso
Use para comunicar um resultado favorável, como uma mensagem de sucesso e botões de confirmação.
Atenção
Use para interface do usuário que comunica cuidado para evitar a ocorrência de um erro ou engano.
Perigo
Use para interface do usuário que comunica perigo ou estados de erro grave.

Paleta

Segue as cores com suas classes e texto contrastantes.
